Pourquoi PlexHubTV

Une seule app pour les power users
multi-source.

Aucune app Android TV ne fait à la fois Plex, Jellyfin et IPTV correctement. Les utilisateurs multi-source jonglent entre trois apps, trois historiques, trois recherches. C'est ce constat qui a fait naître PlexHubTV.

L'app unifie ces sources avec une vraie déduplication multi-serveur (un même film présent sur deux serveurs reste UN film, avec choix de la source au moment de la lecture), une UI Netflix-style pensée pour la télécommande, et un lecteur double moteur (ExoPlayer + MPV) qui ne rend jamais un fichier injouable.

Le projet est porté par une seule personne sur du temps personnel. Le code est en architecture stricte (Clean Architecture multi-module Kotlin, Room v64, Hilt, Jetpack Compose). Distribué en sideload aujourd'hui, Play Store à venir. Une donation premium est en préparation pour soutenir le développement long terme.

Profils & contrôle parental

Un profil pour chacun.
Le contrôle pour les parents.

Jusqu'à 5 profils locaux avec avatar emoji. Activez le mode Kids pour filtrer les contenus par tranche d'âge (7+, 13+, 16+, 18+). Verrouillez le changement de profil avec un code PIN — hash PBKDF2, pas du clair chiffré-au-repos. Chaque profil garde son propre historique, ses favoris, sa watchlist, son offset de reprise et ses préférences de langue/qualité — isolés en base de données, pas filtrés à l'affichage.
